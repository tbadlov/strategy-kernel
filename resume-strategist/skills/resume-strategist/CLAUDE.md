# resume-strategist

7-step interactive interview workflow for crafting CTO resumes targeting Series C/D and IPO-track companies.

## Files

| File | What | When to read |
|------|------|--------------|
| `SKILL.md` | Skill trigger phrases, workflow table, resource references | Modifying activation triggers, changing workflow steps, updating reference file pointers |
| `resume_strategist.py` | Python orchestrator driving the 7-step workflow | Modifying step instructions, changing interview questions, updating output formats, adding steps |

## Directories

| Directory | What | When to read |
|-----------|------|--------------|
| `references/` | Evidence-based frameworks and interview playbooks loaded per-step | Adding research sources, updating interview questions, modifying calibration guidance |
