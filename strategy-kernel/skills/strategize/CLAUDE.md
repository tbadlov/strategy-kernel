# strategize

8-step interactive strategy workflow grounded in Rumelt's Good Strategy Bad Strategy kernel framework.

## Files

| File | What | When to read |
|------|------|--------------|
| `SKILL.md` | Skill trigger phrases, workflow table, resource references | Modifying activation triggers, changing workflow steps, updating reference file pointers |
| `strategize.py` | Python orchestrator driving the 8-step workflow | Modifying step instructions, changing interview questions, updating output formats, adding steps |

## Directories

| Directory | What | When to read |
|-----------|------|--------------|
| `references/` | Rumelt framework reference materials loaded per-step | Adding framework knowledge, updating strategy tests, modifying audit criteria |
